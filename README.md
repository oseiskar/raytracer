An OpenCL raytracer
===================

Usage
-------------
	python clray.py scene-filename

(see `python clray.py -h`)

Requirements
-------------
 * Python (2.7)
 * pyopencl (> 2013), tested with 2013.2
 * pygame (for visualization)
 * nympy
 * scipy
 * sympy (for implicit surfaces)
