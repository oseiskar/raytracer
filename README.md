An OpenCL raytracer
===================

Usage
-------------
	python clray.py scene-filename

(see `python clray.py -h`)

Requirements
-------------
 * Python (2.7)
 * pyopencl (>= 2012), tested with 2012.1, 2013.2 and 2014.1
 * pygame (for visualization)
 * numpy
 * scipy
 * sympy (for implicit surfaces)
