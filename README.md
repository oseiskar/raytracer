An OpenCL raytracer
===================

Usage
-------------
	python clray.py [scene-filename-without-.py]

Requirements
-------------
 * Python (v. 2.7 ?)
 * pyopencl (at least v. 2011.1, tested with 2012.1)
 * pygame (for visualization)
 * nympy
 * scipy
 * sympy (for implicit surfaces)
